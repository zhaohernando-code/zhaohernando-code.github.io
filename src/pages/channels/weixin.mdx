# OpenDesk 扫码一键连接微信

本文介绍如何在 OpenDesk 中扫码连接微信。这是通过复用微信官方提供的微信 ClawBot 插件，以及 [OpenClaw @tencent-weixin/openclaw-weixin 插件](https://www.npmjs.com/package/@tencent-weixin/openclaw-weixin) 来实现的。在 OpenDesk 中，我们实现了一个与 OpenClaw Plugin SDK 对齐的兼容层（Shim Layer），从而可以快速复用 OpenClaw 的部分插件生态。

## 1. 首先升级 OpenDesk 到最新版本

首先升级 OpenDesk 到最新版本。在终端中输入 `opendesk` 进入 OpenDesk 之后，输入 `/upgrade` 打开升级窗口，选择最新的 nightly 版本。

![](/assets/images/common/upgrade_nightly.jpg)

## 2. 在 OpenDesk 中安装微信插件并登录微信

其实这比你想象的要更简单。你需要做的只是打开 OpenDesk 然后输入 **帮我登陆微信**。

下边的视频展示了详细的过程，OpenDesk 会加载内置的 Skills，自动安装微信插件并弹出二维码（二维码来自微信官方的连接授权）。

![](/assets/images/channels/weixin/login.gif)

接下来，你只要打开微信扫一扫，扫码进行配置即可。扫码完成后，在微信中就会弹出微信 ClawBot 的界面，就可以跟你的 OpenDesk 对话了。
