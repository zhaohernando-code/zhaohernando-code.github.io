# 文件整理

把混乱的下载目录、桌面文件、项目资料或临时截图交给 OpenDesk，它可以根据你的描述完成归档、重命名、查重和清理。

<img src="/assets/images/gallery/file_organize.jpg" alt="文件整理" />

## 可以交给 OpenDesk 的事

- 按时间、项目、文件类型或内容主题重新整理目录；
- 找出重复、过期、空文件夹或明显无用的临时文件；
- 批量重命名截图、文档、视频和资料文件；
- 根据目录内容生成摘要，方便你快速确认保留或删除；
- 对处理结果给出变更说明，便于你回看和撤销。

## 使用方式

打开 OpenDesk 后，直接把目标目录和整理规则告诉它即可。你也可以先让 OpenDesk 只生成整理计划，确认后再执行实际移动或删除操作。
