# 批量文件处理

当一批视频、图片、表格或文档需要执行相同处理时，可以把规则交给 OpenDesk，让它选择合适的工具连续执行并汇总结果。

<img src="/assets/images/gallery/file_batch.png" alt="批量文件处理" />

## 常见任务

- 批量压缩视频或图片，并输出处理前后的体积对比；
- 按统一规则裁剪、转换格式或重命名图片；
- 扫描目录中的文件，提取元数据并生成表格；
- 对大量文档做分类、摘要或格式整理；
- 将处理过程和异常文件汇总成一份可追踪报告。

## 建议方式

先让 OpenDesk 对样例文件执行一次处理，确认规则后再扩大到完整目录。对于会覆盖原文件的任务，建议要求 OpenDesk 先创建备份目录或输出 dry-run 计划。
