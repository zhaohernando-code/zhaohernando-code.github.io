# 2026.5.14 日构建更新内容

## 新增功能

- 在系统设置界面中支持手动设置模型的上下文窗口大小，方便对于一些非公开命名的模型进行自动上下文压缩;
- 支持菜单中手动设置系统字体；

## 优化与 Bug 修复

- 优化上下文大小的快速估算算法，修复长上下文情况下有时无法自动触发压缩的问题；
- 优化任务记忆，在上下文压缩时对于原始任务和工作流程有更好的跟踪；
- 修复读文件中由于chardet版本不一致导致的文件读取错误；
- 修复部分搜索工具的proxy识别能力，可以自动识别系统默认代理；
- 支持对工具调用的实时解析和纠错，在模型输出的工具调用出现错误的时候识别并回滚错误的工具调用；
- 修复ttask artifacts目录的自动授权问题，避免上下文压缩落盘后再次读取时报权限弹窗；
