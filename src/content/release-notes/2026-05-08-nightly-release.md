# 2026.5.8 日构建更新内容

## 新增功能

- (桌面端) 支持自动化任务（定时任务，周期任务等）在配置时设置消息渠道，这样即便在pc上配置的定时任务也可以指定发送到手机微信/飞书;
- (桌面端) 收到微信/飞书等渠道消息的时候自动切换到该消息触发的新任务页面;
- (桌面端) 支持通过 /use-browser 支持内嵌网页浏览器控制，在右侧工作空间面版区域嵌入任务级浏览器窗口，不依赖系统浏览器；同时支持面板收起时通过画中画模式持续看到被控页面状态。内嵌浏览器与系统浏览器不共享登陆状态和身份，减少用户身份凭证暴露风险；
- (桌面端) 任务右键菜单提供收藏功能，在任务列表下方提供常驻的收藏面板;
- (桌面端) 支持工作空间中点击预览html网页;

## 优化与 Bug 修复

- 优化上下文大小的快速估算算法，修复长上下文的情况下，压缩有时无法自动触发的问题;
- 优化任务记忆，在长上下文场景下对原始任务和工作流程有更好的跟踪和记录效果;
- 支持跨平台通过 Prompt 构建和权限配置来指定临时文件目录，在 HarmonyOS 设备上使用 `.opendesk-tmp` 作为专有临时目录，避免污染用户家目录 (当前鸿蒙终端默认临时目录为 $HOME);
